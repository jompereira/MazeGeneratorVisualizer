# Maze Generator Visualizer

https://jompereira.github.io/MazeGeneratorVisualizer/

Maze Generator Visualizer is a P5.js project intended to showcase how different maze generation algorithms operates.

This project is inspired by the book MAzes for Programmers by Jamis Buck.

## Algorithms implemented
- Binary Tree
- Sidewinder
- Aldous-Broder
- Wilsons
- Hunt and Kill
- Recursive Backtracker
- Kruskal's
- Eller's
- Prim's
